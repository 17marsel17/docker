2) Запустите контейнер node в интерактивном режиме подключения терминала, поименуйте его mynode, 
передайте две переменные среды NAME=<ваше имя> и SURNAME=<ваша фамилия>

C:\Windows\system32>docker run --name mynode -i -t -e NAME=Марина -e SURNAME=Селезнева node:15.14
Welcome to Node.js v15.14.0.
Type ".help" for more information.

3) В интерактивной среде выполнения node выполните скрипт, который выведет на экран приветствие: 
Привет, <ваше имя> <ваша фамилия>!, эти данные должны быть получены из переменных среды

> console.log(`Привет, ${process.env.NAME} ${process.env.SURNAME}`)
Привет, Марина Селезнева
undefined
>
(To exit, press Ctrl+C again or Ctrl+D or type .exit)
>

4) Остановите контейнер

C:\Windows\system32>docker stop mynode
mynode

5) Удалите образ node версии 15.14 ( нельзя удалить образ, если остались контейнеры. Вероятно пропущен шаг удалить контейнер)

C:\Windows\system32>docker rmi node:15.14
